# Pokedex

Pokedex is a search application built using React and Node.js.

The application allows users to search Pokemon, view details through a simple and responsive interface.

## Features

- Search Pokemon by name
- View Pokemon details
- Responsive UI
- Recently viewed Pokemon

## Tech Stack

### Frontend
- React
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
